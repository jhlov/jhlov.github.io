유니코드 문제가 없다는 파이썬3를 쓰고 있지만 os에 따라서 아래와 같은 에러를 가끔 만날 수 있다.

```
UnicodeEncodeError: 'ascii' codec can't encode character u'\xa1' in position 0: ordinal not in range(128)
```

아래 명령어로 해결 가능

```
$ export PYTHONIOENCODING=utf-8
```
